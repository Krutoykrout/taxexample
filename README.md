# AVTO SELECT — демо-сайт

Статический прототип автомобильного сайта.

## Страницы
- index.html — главная
- vykup.html — выкуп
- avto-pod-zakaz.html — авто под заказ
- catalog.html — каталог
- o-kompanii.html — пустой раздел «О компании»
- otzyvy.html — пустой раздел «Отзывы»
- kontakty.html — контакты
- zayavka.html — форма заявки

## Как показать
Откройте `index.html` в браузере. Верхнее меню ведёт на отдельные страницы.

## Что заменить перед запуском
Название, логотип, телефон, адрес, почту, реальные автомобили, фотографии, юридический текст и обработчик формы.
